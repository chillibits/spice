---
title: Uninstall command
---

The `uninstall` command can be used to delete an installed executable again.

## Usage
=== "Long form"
    Use the `uninstall` command by executing:
    ```sh
    $ spice uninstall [main-source-file]
    ```
=== "Short form"
    Use the `uninstall` command by executing:
    ```sh
    $ spice u [mail-source-file]
    ```
